Setting up Data source in Tomcat for JPA:

1. edit the server.xml with the following code
    <Resource name="jdbc/RestroDB"
              auth="Container"
              type="javax.sql.DataSource"
              driverClassName="com.mysql.jdbc.Driver"
              url="jdbc:mysql://localhost:3306/wisencrazy_restaraunt"
              username="root"
              password="Chhabi@82"
       
              maxTotal="8"
              maxIdle="2"
              minIdle="1"/>

Place this under GlobalNamingResources element

2. Add the below to the content.xml file
<ResourceLink name="jdbc/RestroDB"
                  global="jdbc/RestroDB"
                  auth="Container"
                  type="javax.sql.DataSource" />

3. Add the mysql driver jar file
